# Assign

Assign operator allows to change control properties.
It changes property instantly when steps starts, which means change is not animated, however step duration rules still apply.

### a **\<-** property: value

```dabl tab showLineNumbers
title: 'color and text change'
dot id: d1  text: 'dot' at: [-2,-1] size: 45
step: 'change dot to blue and set new text' duration: 200ms
d1 <- color: blue text: 'tod'
```
```html tab showLineNumbers
<dots-and-boxes controls style="height: 250px" code="
  title: 'color and text change'
  dot id: d1  text: 'dot' at: [-2,-1] size: 45
  step: 'change dot to blue and set new text' duration: 200ms
  d1 <- color: blue text: 'tod'">
</dots-and-boxes>
```
