# Move

move operator allows to move controls

### a **-\>** (x, y)

```dabl tab showLineNumbers
title: 'dot color and size'
dot id: d1  text: 'dot' at: [-3,-1] size: 35
step: 'move dot by 4 cells right'
d1 -> +[4,0]
```
