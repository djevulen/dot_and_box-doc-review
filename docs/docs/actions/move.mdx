# Move

move operator allows to move controls

### a **-\>** (x, y)

```dabl tab showLineNumbers
title: 'move control'
dot id: d1  text: 'dot' at: [-3,-2] size: 35
step: 'move dot by 4 cells right'
d1 -> +[4,0] // relative move
d1 ->  -[0,-1] // relative move
d1 ->  [0,0] // absolute position move
```
